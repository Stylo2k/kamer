Our endpoints are:


  • /properties/
    • GET : returns all properties in the database
    • POST: adds a property to the database
  • /properties/id/{externalId}
    • GET: returns the property
    • PUT: updates the property
    • DELETE: deletes the property
  • /properties/city/{cityName} & {orderBY} & {ascOrDesc} & {minPrice} & {maxPrice} & {pets} & {areaSqm} & {minArea} & {maxArea} & {sqmBudget}
      GET : given a city returns all properties that match the query parameters from the url
    • Order by : order by this attribute example : rent
    • The ordering : Ascending or Descending
    • Rent price range: min, max or both
    • Pets: if pets are allowed
    • Area : min, max or both
    • Square Meter budget: max rent price per square meter maxPrice/maxArea
    • N : number of properties wanted to return Default is 10
  • /properties/location/{longitude} & {latitude}
    • GET : returns a list of all properties that match the longitude and latitude
    • PUT : updates all the properties that match the longitude and latitude
    • DELETE : deletes all the properties that match the longitude and latitude
  • /properties/city/stats/{cityName}
    • GET: returns the stats of a given city
